# Sudoku Solver (Final)

This project is a Sudoku Solver web app using Flask, OpenCV and TensorFlow/Keras.

## Features
- Upload a Sudoku photo → OCR → solve → return solved image.
- Manual entry with animated step-by-step solving.
- Dark/Light theme with persistence, keyboard shortcuts (T, Space), speed persistence.
- Responsive and accessible UI.
